Naming Conventions {
	
	Acronyms are treated as one word 
	N.A.S.A. -> Nasa
	J.S.O.N. -> Json
	S.I.M.D. -> Simd
	L.O.L. -> Lol
	
	One letter acronyms steal the uppercasing of the next letter.
	J. U.T.F8 -> Jutf8
	
	Use a module called "details" for implementation details that need to be exported.
	
	
	myVariable
	MY_CONSTANT_VARIABLE
	myJsonFunction
	myMacroFunction
	
	my_simd_module
	my_crate
	
	MyJsonType
	MyTraitLol
	primitive00
}
